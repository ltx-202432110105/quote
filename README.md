# Logo 使用说明

本仓库包含学士帽 logo 的多种版本，适用于不同背景色的场景。

## 文件说明

| 文件名 | 描述 | 适用场景 |
|--------|------|----------|
| `4fec94165dba3785e5b72973f6013f00.jpg` | 原始源文件（白色背景，黑色线条） | 原始素材，请勿直接修改 |
| `logo-white.png` | 透明背景，**白色**（#FFFFFF）学士帽线条 | 深色背景（深色 PPT、深色网页、深色海报） |

## 使用场景对比

- **浅色背景**（白色/浅灰 PPT、网页）：使用黑色线条版本（直接使用原始 JPG 或自行生成黑色线条透明 PNG）。
- **深色背景**（深色 PPT 幻灯片、黑色/深灰网页、夜间模式）：使用 `logo-white.png`，学士帽线条为纯白色，背景完全透明。

## logo-white.png 预览

> 以下预览在深色背景下可正常显示白色线条；在浅色背景（如 GitHub 默认）中线条不可见属正常现象。

在 PPT 或网页中插入 `logo-white.png` 时，建议置于深色底色块上以获得最佳效果。
